Send App Log

Use this to send information about your phone and, if you have a Bluetooth connection, about your GPS device.  The file normally available is MTKutilityLog.txt.

MTKutilityErr.txt is created when MTKutility encounters an unrecoverable error.  The app is coded to catch these errors and restart itself with a popup to send the file to me.  Please send the MTKutilityErr.txt file when this happens as this usually reflects a problem that has to be fixed.

Both files will give me information about your phone and, if you connect before sending, the GPS device's information string.  Knowing device combinations helps with problem resolution and future enhancements.  There is no personal information in the file and you can verify this by looking at a file before you send it.
