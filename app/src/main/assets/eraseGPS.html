Erase Log

This function starts by re-reading the log record count to determine if there is anything to erase.  If there is the Erase GPS Records button is enabled.  Press it to start.  The result is a full format of the GPS memory allocated to track point storage.

dealing with problems that arise
The memory formatting should complete in under a minute.  If it extends to two or more minutes something went wrong.  Stop the function by using your phones back button, then force close MTKutility.  Force closing is faster than trying to back out through the Main screen.  You will need to turn your logger off and on again, then run MTKutility and connect to your logger to ensure that you logger is properly reset.
