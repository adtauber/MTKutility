Welcome to MTKutility 5.06

This app will only work with a GPS device has the MTK GPS chip
Version 5.06 changes
Fixed GPX output to align with GPX 1.0 standard and enable track viewing with Garmin Basecamp.  Modified KML output to reflect the track structure of the GPX file.

VERY IMPORTANT: please read the Help documents.  MTKutility failures can leave your GPS device non-functional.  The help describes how you can recover from failures and also how you can use preferences to change behavior and potentially overcome app issues.

The activities that appear in the menu are driven by your GPS logger capabilities. While Help documentation shows all the activities, the menu you see after you connect will reflect your GPS device's capabilities. 

I created this app for personal use and I am not able to provide dedicated time to dealing with issues.  However, I try to resolve issues in my spare time.  Please email the processing log if you experience an app abort.

You can also email adt.androidapps@gmail.com, or better still, use the Send App Log menu item after you have connected to your GPS and add your comments to the generated email.  That way I will also have a bit of information about your devices.

Version 5.05 changes
Fix European number format issues in binary to GPX/KML/CSV conversions.

Version 5.04 changes
Fix binary conversions time stamp.  Track point date and time was being set to local time.  Has to be UTC time.

Version 5.02 changes
Partial rewrite of binary conversions to resolve iBlue 747 issues.

Version 5.01 changes
Fix Holux M-241 binary file processing issue.

Version 5.0 changes
Fix KML file <Placemark> bug.

Version 4.9 changes
Bug fixes for KM file format. Time was missing when milliseconds was not recorded.  Track point data was not being generated for all source files and a couple of files had mismatched XML tags.

Version 4.8 changes
Binary file conversion expanded to GPX, KML and CSV file formats.

Version 4.7 changes
Enhanced GPX file creation to include way points. GPX file XML matches GPSbabel structure and includes all the XML tags created by BT747.

Version 4.6 changes
Users with a Holux M-241 can now create GPX files with MTKutility.  The GPX creation module has also been enhanced to offer the same XML output that GPSbabel provides. Now you can use and see the result for course, fix, satellites in view, hdop, vdop, and pdop.  Also added a sequential track point name so that you can verify that all records stored in your GPS are accounted for in the GPX file.
