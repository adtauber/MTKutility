Get EPO file

Use this screen to down load an Extended Prediction Orbit (EPO) file for your AGPS data update.

There are 2 flavors of EPO data, 60 byte data SET and 72 byte data SET.   MTKutility currently has 2 sites available for EPO data -  the Qstarz 60 byte site, and the Globaltop 72 byte site.   It is delivered with the Qstarz site selected.  Use the Select FTP URL button to change sites and the Save Settings button to save the change for future use.  You will need to rely on trail and error if you do not know which site to pick.  My Qstarz logger will accept 72 byte records without error but it does not store the EPO data - and it takes forever (hours) to complete.

The app preference AGPS memory size - days is associated with AGPS updates.  Your logger will have a maximum size for stored records. I believe most loggers store 7 days but the sure way to know is update the AGPS using your PC application.  Then note the EPO set value on the Home screen after you connect, divide this by 4, and set the result in your preferences.

The Globaltop 72 byte site has 10 files, each with 3 days EPO data to make up a 30 day set.  Updating with successive files will only result in 3 days data being stored on your GPS.  You have to join files to get a 7, 14 or 30 day AGPS update.  Here's how.  Download the EPO_GPS_3_1.DAT file.  Then click the Select Local File button and choose this file. It will appear in the local file name slot and is the file you will append to.  Then click the Select EPO File button again and select the EPO_GPS_3_2.DAT file.  This will unlock the Append EPO file button.  Click it to do the append.  Repeat with the Select EPO File - Append EPO file buttons combination until you have enough days collected to do a full AGPS update.

The Globaltop 72 byte site also has 60 byte files (MTK7d.EPO, etc.) but the updates have not been reliable.

You can add sites with the Add FTP site button.  You will need the FTP IP and the  FTP userid and password.  GPS merchants do not normally publish this information in their documentation so you will need to use network traffic capture on your PC to find the values.  The tool I used for this is Wireshark.  I documented the procedure I used, including screenshots, so that I don't have to relearn every time.  Please send an email requesting this document if your logger isn't a Qstarz and you are willing to try to get the FTP login info.  Use Remove FTP site  to remove an item from the saved FTP site list.
