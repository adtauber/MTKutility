Make GPX file

Use this screen to create GPX files from previously downloaded binary files.  You do not need to connect to your GPS logger to use this.  You have to select the binary file you want to decode before you can create the GPX file.  The file browser defaults to the MTKutility/bin location but you can navigate to other locations.  You should be able to read a file from any user accessible storage location on your phone including your SD card.  Your phone's security will determine what you can do.  The GPX file is created in MTKutility/gpx.

The file browser has been designed to let you navigate folders by clicking folder names.  Files are selected by clicking the file graphic at the left of the file name.  Files that do not have sel in the graphic cannot be selected.

NOTE: The GPX file should have enough data for geotagging pictures but it may not have all the data you selected for the log.  Use BT747 to create your GPX files if you use more than the default settings for your logger.
